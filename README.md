# My Recipe Book

This folder is ready to upload to GitHub Pages.

## Structure

- `index.html` — the website
- `recipes/index.json` — list of recipe files
- `recipes/*.json` — one file per recipe

## Add a recipe

1. Copy an existing file from the `recipes` folder.
2. Rename it using lowercase words and hyphens, for example:
   `chicken-parma.json`
3. Edit the recipe details.
4. Add the new filename to `recipes/index.json`.

Example:

```json
[
  "creamy-tuscan-chicken.json",
  "roasted-pumpkin-pasta.json",
  "sunday-banana-bread.json",
  "chicken-parma.json"
]
```

Do not add a comma after the final filename.
